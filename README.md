# 📚 Course Management System

A full-stack **MERN (MongoDB, Express, React, Node.js)** application designed to manage courses, users, and learning materials efficiently.

---

## 🚀 Features

* 👤 User Authentication (Login / Register)
* 📖 Course Creation & Management
* 🎓 Student Enrollment
* 📂 Upload & Access Course Materials
* 🧑‍🏫 Instructor Dashboard
* 📊 Role-based Access Control (Admin / Student / Instructor)

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Tools

* Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/course-management-system.git
cd course-management-system
```

---

### 2️⃣ Install dependencies

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

---

### 3️⃣ Environment Variables

Create a `.env` file in backend folder and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

### 4️⃣ Run the project

#### Backend

```bash
npm run dev
```

#### Frontend

```bash
npm run dev
```

---

## 🔗 API Endpoints (Sample)

* `/api/auth` → Authentication
* `/api/courses` → Course management
* `/api/enrollments` → Student enrollments
* `/api/materials` → Course materials
